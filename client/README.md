# Anti-Soy Frontend

## Installation
```bash
cd client/
npm i
npm run dev
```

## Environment

Create a `.env` file in `client/`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Technologies used for the frontend.

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
